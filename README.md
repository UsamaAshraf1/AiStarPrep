# AI-StarPrep

AI-StarPrep is a modern web application built with React and Vite, designed to provide AI-powered preparation tools. This project includes various integrations, such as Stripe for payments, Redux for state management, and form handling with Formik and React Hook Form.

## Features
- **Vite-powered development** for fast builds and hot reloading.
- **State management** using Redux Toolkit.
- **Stripe integration** for payment processing.
- **Form validation** with Formik and Yup.
- **Data export/import** using json2csv and papaparse.
- **PDF generation** with jsPDF and AutoTable.
- **Charts & graphs** using Recharts.
- **Tailwind CSS** for styling with additional scrollbar utilities.

## Installation

Ensure you have Node.js installed. Then, clone the repository and install dependencies:

```sh
npm install
```

## Development

To start the development server:

```sh
npm run dev
```

## Build

To create a production-ready build:

```sh
npm run build
```

## Linting

To check for linting issues:

```sh
npm run lint
```

## Preview

To preview the production build:

```sh
npm run preview
```
